Wepa Recipes

Flatiron Project #4

About this project:

Rails API backend, JavaScript frontend. MVC and RESTful conventions. All interactions between the client and the server are handled asynchronously (AJAX) and use JSON as the communication format. Uses Object Oriented JavaScript (classes) to encapsulate related data and behavior.

A user can post a recipe and a description of it along with the ingredients. They can also edit the recipe and delete it.


Why:

My family is Puerto Rican and I always ask them for various different recipes because they’re just too delicious and I’d love to share them with the world. With that in mind, I dedicated this project to that idea. The recipes and ingredients can be added and located in one place.
