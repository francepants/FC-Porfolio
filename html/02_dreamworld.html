Dream World

API, CLI

About this project:

Built using a domain specific language (DSL) called Sinatra, along with models, views, controllers (MVC).

This is a site where a user is able to sign up and/or log in and upon doing so, they’ll be able to post entries of their dreams that are only accessible to them. They have the option of editing the dreams and deleting them as well.
