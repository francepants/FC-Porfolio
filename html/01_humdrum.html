Humdrum

Flatiron Project #1

What it is:

This is a Command Line Interface (CLI) app in Ruby that uses Bored API to list 10 random activities (the data is stored as objects) for a user to do when they’re bored. When listed, the user will see the activity, how many participants are required, and the type of activity (relaxation, recreational, busywork, etc).


The CLI Interface:

The CLI prompts the user for an input. Here, it’s where they’d like to “list” out the activities, or end the session with “kbye”. If they choose to list the activities, a random assortment of 10 activities will be presented. Then the user can choose to go back and get another random 10 activities or if they’ve happily found a cool activity, they can exit.

Why:

I chose this API because it was during the beginning of the Covid pandemic. We were in the house, bored, unsure what to do with our time. I thought this would be a fantastic idea to cure the idea of a bummer summer that was approaching, and turn it into one full of (socially distanced) activities.
